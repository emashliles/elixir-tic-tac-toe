# Elixir Tic Tac Toe

https://travis-ci.org/emashliles/elixir-tic-tac-toe.svg?branch=master

## Running the game

   You can run the game using the mix task included. 

    ```elixir
    mix compile
    mix tic_tac_toe
    ```

## Run the tests continuously

    ```elixir
    mix test.watch
    ```
